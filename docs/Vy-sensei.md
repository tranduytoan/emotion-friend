## Prompt tạo ảnh khuôn mặt cô Vy

Sử dụng prompt này để tạo bộ ảnh khuôn mặt nhất quán của cô Vy cho hệ thống Emotion Friend. Mục tiêu là tạo cùng một nhân vật, chỉ thay đổi biểu cảm giữa các trạng thái cảm xúc.

```text
Create a consistent Vietnamese female teacher character named "Cô Vy" for a children's emotional learning app.

Character specification:
- Age: late 20s to early 30s
- Warm, gentle, trustworthy teacher-like appearance
- Friendly oval face, soft skin tone, expressive eyes, natural eyebrows
- Medium-length dark hair, neat and tidy
- No glasses unless explicitly requested
- Clothing should be simple, pastel, child-friendly, and consistent across all images
- Head-and-shoulders portrait framing, centered composition
- Clean background, soft pastel colors, no distracting objects

Generate separate face-expression variants of the same character with these emotions:
1. Happy — bright smile, warm eyes, relaxed cheeks
2. Sad — gentle teary eyes, downturned mouth, compassionate expression
3. Angry — slightly furrowed brows, firm lips, controlled anger, not scary
4. Surprised — wide eyes, slightly open mouth, natural surprise
5. Calm — soft neutral smile, relaxed face, peaceful presence
6. Tired — slightly droopy eyes, subtle fatigue, still kind and caring

Style requirements:
- Illustration should look polished, modern, and suitable for a children's app
- Soft lighting, smooth linework, high clarity, clean facial features
- Keep hairstyle, face shape, outfit, and identity identical across all expressions
- Only the facial emotion should change
- Use a consistent visual style for the whole set

Avoid:
- Extra characters
- Text, watermark, logo
- Overly dramatic or scary facial expressions
- Different hairstyles or clothing between images
- Low-quality, blurry, distorted face anatomy

Suggested output:
- 1 square portrait for each emotion, preferably 1024x1024
- Transparent background or very light pastel background
```

### Negative prompt

```text
blurry, low resolution, distorted face, deformed hands, extra fingers, scary expression, harsh shadow, heavy makeup, text, watermark, logo, duplicate face, inconsistent character, different hairstyle, different clothes, cluttered background
```
