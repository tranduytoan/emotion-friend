# Story Images

Each story folder must contain exactly 4 files:

- `1.jpg`
- `2.jpg`
- `3.jpg`
- `4.jpg`

If `story_images.folder_name = "topic_1_story_a"`, files must be placed at:

- `res/img/stories/topic_1_story_a/1.jpg`
- `res/img/stories/topic_1_story_a/2.jpg`
- `res/img/stories/topic_1_story_a/3.jpg`
- `res/img/stories/topic_1_story_a/4.jpg`

Runtime URL format:

- `/static/stories/{folder_name}/{index}.jpg`
