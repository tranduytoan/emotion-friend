# Story Images

Each story folder must contain exactly 4 files:

- `1.png`
- `2.png`
- `3.png`
- `4.png`

If `stories.image_folder = "story-1"`, files must be placed at:

- `res/img/stories/story-1/1.png`
- `res/img/stories/story-1/2.png`
- `res/img/stories/story-1/3.png`
- `res/img/stories/story-1/4.png`

Runtime URL format:

- `/static/stories/{folder_name}/{index}.png`
